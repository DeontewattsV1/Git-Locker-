# Privacy Redactions

This repository contains a public-safe development history, not a raw transcript dump.

## Redacted or omitted from public history

The following categories were intentionally omitted:

- Personal phone numbers
- Private email/account access details beyond public project links
- Authentication, dashboard, or account-management details
- Raw pasted text containing sensitive or irrelevant operational fragments
- Any private information not needed to understand or build Git Lockers

## Why this matters

Git Lockers is public. Public repositories should not expose private contact information, account metadata, secrets, tokens, access credentials, or unnecessary personal details.

## Rule for future commits

Before committing generated chat history or prompt logs, check for:

- phone numbers
- personal addresses
- private emails not intended for publication
- API keys
- access tokens
- account dashboards
- unpublished business information
- third-party private content

When in doubt, summarize instead of copying raw text.
