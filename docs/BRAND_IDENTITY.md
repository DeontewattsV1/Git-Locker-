# Git Lockers Brand Identity

## Core concept

**Git Lockers** is an interactive developer vault where builders store, unlock, remix, and activate repository ideas, prompt capsules, GitHub Actions, workflow patterns, and agent-ready systems.

## One-line pitch

> Git Lockers is an interactive vault for reusable GitHub ideas, workflows, prompt capsules, and agent-ready build systems.

## Primary tagline

> **Unlock the next repository.**

## Brand personality

| Trait | Meaning |
|---|---|
| Secure | Feels like a vault, locker room, archive, or protected builder system |
| Interactive | Users explore, open, filter, activate, and expand |
| Agent-native | Designed for ChatGPT, coding agents, GitHub Actions, and workflow automation |
| Builder-first | Practical, repo-ready, copy-pasteable, modular |
| Open-source expandable | Others can add lockers, prompts, workflows, and templates |

## Naming system

| Brand Element | Meaning |
|---|---|
| Git Lockers | Full interactive environment |
| Locker | One stored repo idea, workflow, prompt pack, or build capsule |
| Key | A command, prompt, trigger, or GitHub Action that unlocks a workflow |
| Vault | A collection of lockers by category |
| Capsule | A structured prompt or workflow instruction |
| Workbench | Area where users generate, remix, or configure a locker |
| Action Bay | GitHub Actions automation center |
| Audit Room | Safety, validation, citations, and policy checks |
| Launch Door | Product Hunt, GitHub release, or public publishing path |

## Voice system

Say:

- Unlock
- Validate
- Ship
- Fork
- Remix
- Run
- Audit
- Open the locker
- Build from this locker

Avoid:

- Magic
- Revolutionary without proof
- Guaranteed first-ever
- AI-powered everything
- Unsafe automation claims
