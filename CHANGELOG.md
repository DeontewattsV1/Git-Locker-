## [1.0.0] - 2026-05-19

### Added

- 1,000 foundational Git repository concept seeds (CSV + JSON + schema)
- 1,000 prompt capsules with 20 category docs
- Validation scripts: validate-locker.py, validate_prompt_capsules.py, build_prompt_index.py
- GitHub Actions CI workflows for automated validation
- PR template, issue templates, and contribution guidelines
- Brand assets: logo SVG/PNG, hero banner, social preview
- Interactive README with capsule-render, typing SVG, mermaid diagrams, stats cards
- Fail-closed prompt-injection detection in capsule validator
- Testing skills for data integrity and README rendering validation
- CONTRIBUTING.md, CODE_OF_CONDUCT.md, SUPPORT.md, CLAUDE.md
