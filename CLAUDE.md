# CLAUDE.md

## Project

Git Locker is an open idea locker of foundational Git repository seeds.

## Agent behavior

When helping maintain this repo:

1. Preserve CSV field order.
2. Keep repo names lowercase and hyphenated.
3. Never claim verified novelty unless evidence is included.
4. Prefer small, composable primitives over broad app ideas.
5. Keep contribution paths simple for first-time builders.
6. Validate data before proposing commits.
