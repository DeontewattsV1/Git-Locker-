## Getting Help

- Open an [issue](https://github.com/DeontewattsV1/Git-Locker-/issues) for bug
  reports or feature requests
- Use the issue templates for structured proposals
- Check existing issues before opening duplicates

## Security

If you discover a security vulnerability, please report it responsibly via
GitHub's private vulnerability reporting feature rather than opening a public
issue.
